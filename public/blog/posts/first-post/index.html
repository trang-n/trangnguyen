First Blog Post

Trang Nguyen 2021-04-04 1 minute read

My first blog post! I created this website so I can keep track of my hobbies including books and games. I also want to write about different subjects such as statistics, R, Python, and languages as I learn more about them.

This website was built in R using the blogdown package and Hugo, deployed and hosted via Netlify. I chose a minimalist theme called Renga, developed by Nan Xiao.

I’m new to web development and would love to learn from others on how improve/customize my website.
